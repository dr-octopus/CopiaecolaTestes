Java Web Start

O link aponta para o arquivo de configurações .jnlp (Java Network Launcher Protocol) e será iniciado o download do mesmo. Ao executar o arquivo, será requisitado o .jar correspondente para o servidor no caminho apontado pelo .jnlp. Então o mesmo verificará caso haja uma nova atualização para o .jar e o baixará, em seguida, o .jar será executado.
